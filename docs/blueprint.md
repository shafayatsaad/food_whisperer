# **App Name**: FoodSahayak

## Core Features:

- SMS/WhatsApp Interface: Allows donors, shelters, and drivers to interact with the system via SMS/WhatsApp for maximum accessibility.
- AI Matching Engine: Matches available food surplus to registered shelters based on a Need Score that factors in recency of donation, capacity, and demographics.  The LLM acts as a 'tool' to determine how the parameters for Recency, Capacity and Demographics get changed during the automated run.
- Safety Gate: Implements a safety check to ensure that all donations meet food safety standards before dispatch.
- Donor Flow: Simplifies the donation process with quick SMS/WhatsApp commands for donors to report available food.
- Shelter Flow: Provides shelters with simple SMS/WhatsApp alerts to accept or reject food donations.
- Driver Flow: Enables drivers to accept or decline rescue missions via SMS/WhatsApp and report any issues encountered.
- Admin Dashboard: Provides a web-based dashboard for organization staff to oversee the system, manage users, and manually intervene when necessary.

## Style Guidelines:

- Primary color: Vibrant green (#2ecc71) to reflect freshness and sustainability, evoking Bangladesh's lush landscapes.
- Background color: Light off-white (#f9f9f9) to provide a clean and modern backdrop, allowing content to stand out.
- Accent color: Warm orange (#e67e22) to highlight important actions and alerts, reminiscent of Bangladesh's rich cultural heritage.
- Font pairing: 'Poppins' (sans-serif) for headlines and 'PT Sans' (sans-serif) for body text, providing a contemporary yet readable interface.
- Use clean, minimalist icons representing food types, shelters, and transportation. The icons should have a touch of Bangladeshi motifs and design elements.
- A clean and modern layout that emphasizes ease of use, focusing on clarity and simplicity for all user interfaces, especially the admin dashboard.
- Subtle transitions and animations to provide feedback on user interactions, such as accepting a donation or confirming a pickup, enhancing the user experience without being distracting.
